Website Design Using WordPress (Systems Analysis and Design Class)

URL: two.parkside.team

Myself and five others were in charge of building this website for the Women’s Resource Center of Racine. Throughout the semester, we used WordPress to develop and refine the site, making weekly changes based on feedback. Database Design for WRC Donations (Systems Analysis and Design Class)

I personally created the database for the WRC's donation system using Microsoft Access. I followed the WRC's guidelines for what information they record for donations and built the database accordingly. This involved structuring the tables, defining relationships, and creating forms and reports to streamline the data entry and tracking process.
